<h1 align="center">📄 简历优化器</h1>
<h3 align="center">粘贴你的简历 + 目标岗位JD → 自动提炼亮点、优化表达、匹配关键词</h3>

<p align="center">
  <i>"同样的经历，不同的写法，面试邀约率翻倍"</i>
</p>

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 🔍 **JD匹配** | 对照职位描述，找出你简历里缺少的关键词 |
| ✨ **经历提炼** | 把你写的流水账变成"成果导向"的专业描述 |
| 📏 **量化升级** | 在合适的维度上加数据，让成果可衡量 |
| 🎯 **一简历一岗位** | 根据不同的JD，生成定制化的简历版本 |

---

## 🚀 快速开始

```bash
git clone https://github.com/sissc0731/resume-boost.git
cp -r resume-boost/.claude ./
```

然后说：

> "这是我的简历 [粘贴简历]，帮我针对这个岗位优化：[粘贴JD]"
